尝试写md文件喵~

2024/9/27 learn Markdown 基础语法

QwQ

  • 我只写一些我需要的事情捏~

    1. 标题：

      • 标题使用#作为启始标记，几级标题就是几个#。

    2. 段落：

      • 不要在段落前空格或者制表符（tab）。

    3. 换行：

      • 连续两次enter。
      • 行后两个空格。

    4. 强调语法：

      • 粗体
        **enter**->enter
        __enter__->enter
      • 斜体
        *enter*->enter
        _enter_->enter
      • 混合
        ***enter***->enter
        ___enter___->enter

    5. 段落块引用：

      • >
        块引用可以嵌套
        >qwq

        qwq

    6. 列表：

      • 有序列表
        1. qwq
        2. awa

        注意1.与qwq之间有空格

      • 无序列表

        1. *
        2. +
        3. -

          一个表内建议使用同一个符号制表捏

      • code blok

        • 通常使用四个空格或一个制表符缩进，
        • 在列表中时使用八个空格或两个制表符缩进。

          tip:需要和上一行语句空出一行

            main()  
  {
      for(int i==0,i<10,i++)
      {
          if (i==5)
          {break;}
      }
  }

    7. 代码语法：

      • qwq
        两个反引号
        `qwq`->qwq

    8. 分割线：

      • ***
      • ---
      • ___

      为了兼容性请在分割线前后添加空白

    9. 链接语法：

      • [markdown](https://markdown.com.cn "markdown教程链接")
        markdown

    10. 图片语法:

      • 图片
        ![图片](/assets/img/philly-magic-garden.jpg "Magic Gardens")
      • 链接图片
        [![图片](/assets/img/shiprock.jpg "Shiprock")](https://markdown.com.cn)

    11. 转义字符：

      CharacterName
      \ backslash
      ` backtick (see also escaping backticks in code)
      * asterisk
      _ underscore
      { } curly braces
      [ ] brackets
      ( ) parentheses
      # pound sign
      + plus sign
      - minus sign (hyphen)
      . dot
      ! exclamation mark
      | pipe (see also escaping pipe in tables)

    12. markdown支持内嵌html标签喵：

      直接写html标签就行惹喵，最好空行写html捏~
